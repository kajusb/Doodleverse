# Doodleverse

Turn a hand-drawn paper map into an explorable 3D world in the browser.

## Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Three.js + React Three Fiber
- Gemma 4 (via Gemini API) for sketch interpretation
- MongoDB Atlas for world persistence

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.
